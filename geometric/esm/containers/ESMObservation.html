Create esm observation container

Parameters

  • img_meas – dict of ESMImageMeasurement objects, with keys for camera names.

  • agent_rel_mat (array) – The pose of the agent relative to the previous pose, in matrix form [batch_size, timesteps, 3, 4].

  • control_mean (array, optional) – The pose of the agent relative to the previous pose, in rotation vector pose form. Inferred from agent_rel_mat if None. [batch_size, timesteps, 6]

  • control_cov (array, optional) – The convariance of the agent relative pose, in rotation vector form. Assumed all zero if None. [batch_size, timesteps, 6, 6].
