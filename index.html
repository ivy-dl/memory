End-to-end memory modules for deep learning developers, written in Ivy.

Overview¶

What is Ivy Memory? [coming soon!]

Ivy memory provides a range of differentiable memory modules, including learnt modules such as Long Short Term Memory (LSTM) and Neural Turing Machines (NTM), but also parameter-free modules, such as End-to-End Egospheric Spatial Memory (ESM).

The library is built on top of the Ivy deep learning framework. This means all memory modules simultaneously support: Jax, Tensorflow, PyTorch, MXNet, and Numpy.

A Family of Libraries

Ivy memory is one library in a family of Ivy libraries. There are also Ivy libraries for mechanics, 3D vision, robotics, and differentiable gym environments. Click on the icons below for their respective github pages.
